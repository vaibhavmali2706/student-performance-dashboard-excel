# student-performance-dashboard-excel

Student Performance Dashboard (Excel + VBA):-
An interactive Excel dashboard designed to analyze student performance across a 3-day course using KPIs, PivotTables, slicers, and VBA automation.

Features :-
Dynamic KPI Cards (Average Score, Highest Score, Lowest Score, Target Hit %).
Interactive PivotTables & slicers.
VBA automation to control slicers using checkboxes (1,2,3,4).
Automated filtering & synchronization.
Clean dashboard UI with professional layout.
Real-time updates on slicer interaction.

Tools & Skills Used :-
Microsoft Excel
Excel Formulas
PivotTables & PivotCharts
KPI Card Design
Conditional Formatting
VBA (Macros & Automation)
Dashboard Design & Layout

VBA Automation :-
This project includes VBA macros that:
Connect checkboxes to slicers
Dynamically filter PivotTables
Improve dashboard interactivity
Reduce manual filtering steps
